👤 Үйлчлүүлэгч (customer)

  • Арга хэмжээний төлөвлөгөө
  • Хүслийн жагсаалт

🏢 Үйлчилгээ үзүүлэгч (provider)

  • Өөрийн байршил, багц удирдах

  • Цагийн хуваарь, үнэ
  • Захиалга харах

  • Хэрэглэгчийн удирдлага
